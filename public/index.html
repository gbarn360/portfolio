Testlet

side project | personal

Created a web application similar to Quizlet, designed for self-study and self-quizzing on user-selected topics. The platform allows users to structure their learning by subjects, enabling them to create, edit, and delete flashcards. Personalized quizzes can be generated, with options for various question formats like multiple choice, fill-in-the-blank, and true/false. After completing quizzes, users receive feedback to aid in their ongoing improvement. The project was developed using technologies including React.JS for the frontend, Tailwind for styling, and PHP with MYSQL for backend functionalities.
